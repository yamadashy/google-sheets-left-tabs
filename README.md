# Move Google Sheets tabs on top.

![](./promo/Screenshot_1280x800.png)

## Install

[Google Sheets tabs on top - Chrome Web Store](https://chrome.google.com/webstore/detail/sheets-tabs-on-top/lbhlhhckfpdpafckdiklcbamkmogjdjc)

# Development
- Node.js v10.16.0

## Setup
```
$ yarn
```

## Watch and build
```
yarn dev chrome
yarn dev firefox
yarn dev opera
yarn dev edge
```

## Build extension
```
yarn build chrome
yarn build firefox
yarn build opera
yarn build edge
```

## Lint codes
Lint only.
```
yarn lint
```

Lint codes with fix.
```
yarn lint:fix
```

## Type Checking
```
yarn type-check
```

## Show supported browsers
```
yarn supported-browsers
```
